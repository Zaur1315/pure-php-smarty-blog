# Pure PHP Smarty Blog

Simple blog application built with pure PHP, MySQL and Smarty.

---

## Tech Stack

- PHP 8.1+
- MySQL 8
- Smarty
- SCSS
- Docker
- No PHP frameworks

---

## Quick Start

```bash
docker compose up -d --build
```

Application:
http://localhost:8080

phpMyAdmin:
http://localhost:8081
